## Introduction
The goal of this project is to design and implement an Android-based second-hand trading platform on campus. The project was completed by a three-member team. I was mainly responsible for database design and maintenance, and back-end server design and implementation.
## Link
Back-end API repository link: https://gitee.com/alyssa-zhou/secondhand
Android App repository: https://gitee.com/zhang-zhenwei99/swjtu_secondhand
## Design and Implementation  
### Database
- Using MySQL to implement
- Deploying to Raspberry Pi
### Server
- Backend API design follows REST principles
- Using Java and Spring Boot to develop
### Contribution
- Cooperated with the front-end developer to debug. 
- Created and updated documentation in support of development efforts.
- Participated in code reviews to make sure the code is well-designed, self-documented, testable, secure, and efficient.
